# App 5: Electric Bill Forensics Calculator

This app helps show homeowners how their electric bill is built.

## Core Formula

Total Cost = (Monthly kWh Used x Price per kWh) + Fixed Monthly Fees

## Features

- Monthly kWh usage slider
- Price per kWh slider
- Fixed delivery/program fees slider
- High usage scenario slider
- Current bill estimate
- High usage bill estimate
- Effective all-in rate per kWh
- Bill composition breakdown
- Usage sensitivity table
- Monthly kWh vs bill chart

## Run

streamlit run app_5_electric_bill_forensics_calculator.py
